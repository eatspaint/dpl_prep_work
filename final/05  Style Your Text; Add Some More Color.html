Styling Text, Setting Backgrounds & Gradients

You should now be able to build a basic page, add some minimal CSS, and layout the contents of the page. In this section, we will cover your options when it comes to styling text, as well as setting backgrounds and gradients.

  • Change text properties like: alignment, decoration, indentation, & shadow

  • Set gradient and transparent backgrounds
  • Add background images to elements
  • Set the new CSS3 properties of a background-image
