The Box Model & Positioning

You’ve learned how to get everything on the page, but odds are, if you set out to make a website with what you know now - it would look pretty boring. One of the most important parts of making a good website is knowing where to put things. Luckily, there are more than a few ways to do this with HTML & CSS.

After this lesson, you should know: 

  • What the box model is, and how to use it
  • How to change the size of elements
  • How to add margins, padding, and borders to elements and how to manipulate these properties
  • The different ways to position content on a page: floats, block/inline-block, absolute/relative positioning
  • How to work with the different display types available: clearing floats, killing whitespace, unique positioning
