Before We Start

Ruby uses a lot of different symbols and syntax than HTML/CSS, so lets get started by warming up with Typing.io's Ruby on Rails Lesson.

Learning Ruby, The Hard Way

Zed Shaw’s Learn Ruby the Hard Way has become a staple for beginners learning Ruby and actually learning it. This will be the first resource that we walk through to start learning the basics of Ruby.

Zed has a lot of great things to teach beginners about Ruby, but we have a few notes to make before sending you his way:

  • Zed is big on using a bare-bones editor that doesn’t do much for you. We do not recommend that you use TextWrangler. When he talks about TextWrangler, cover your ears. Use Atom or Sublime instead.
  • Do the exercises! Students who show up to the first day of class with a ton of example files saved to their machine from this prep work tend to be the ones who do better in the class.

After this section, you should know:

  • How to set up a Ruby file, run it, and use the Ruby console, IRB (pronounced 'eye-are-bee', not ‘herb’)
  • How to put/print output in the console
  • How to write comments in Ruby code
  • How to use mathematical operators
  • How to name and store variables in Ruby
  • How to work with strings and escape characters
