15: Loops & Arrays

Storing & Working with Lists of Data

Now that you can make decisions in your code, you should know how to make those decisions repeatedly, over a whole list of information. Ruby arrays are ordered lists of data that can be accessed all from one container (the array). With an array, your code can now loop over a whole list of information and perform some logic to each element in the array.

After this lesson, you should know:

  • How to create an array in Ruby
  • How and when to use different types of loops in Ruby: for, while, each
  • How to build an array with a loop
  • How to use logic conditions with a while loop
  • How to access array elements in Ruby
  • How to build a program with branches
