01: Unix / Command Line

The Command Line

This section will focus on teaching you the fundamentals of navigation and file handling in a Unix-like system. After this tutorial, you should understand:

  • Why we call Mac OS X “Unix-like” and not just “Unix”
  • The layout of your terminal
  • How to create, navigate, and destroy files and directories using the command line
  • Basic Unix terminology and commands i.e. root, ls, pwd, cd, mv, etc.

Before getting started, you should know how to get to your Terminal in OS X. There are two fairly basic ways to open your terminal:

  1. From Finder: Open your Applications folder, and navigate to Utilities, click on Terminal (the black square icon with ‘>_’ inside) Link with Pictures
  2. From Spotlight: Press command-space on your keyboard, and type ‘Terminal’ into the field that comes up, hit enter when you see the icon for ‘Terminal'

Before You Start

Having an adequate typing speed is cruicial to your success in this life. Before each lesson, we will begin with a typing exercise in a language that pertains to the content, to get you warmed up for that lesson.

The command line is generally interfaced with in a language called Bash, so we will begin with Typing.io's Bash Git Exercise. This is probably not going to be comfortable to type at first. Bash, and most other programming languages, rely heavily on symbols as part of their syntax.

Wrapping Up

You should now:

  • Be comfortable navigating your machine from the command line
  • Be able to move, copy, create, & delete files from the command line
  • Be able to find the contents of a directory, the current user’s name, and the current directory from the command line

If the content here feels confusing, that’s ok. Command line functions will be solidified as you use them daily in the life. You can come back to this section at any time to review the commands, or to read the more advanced content.

Quiz

  4. How would you make a new directory?

  6. What command will show you your current directory?

  8. What is the short hand command to navigate to your home directory?

  10. What symbol represents the ‘root’ directory of your computer?
